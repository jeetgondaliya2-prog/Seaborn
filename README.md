# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/jeetgondaliya2-prog/Seaborn.git
```

Move into the folder

```bash
cd Seaborn
```

Install required libraries

```bash
pip install seaborn matplotlib pandas numpy jupyter
```

Launch Jupyter Notebook

```bash
jupyter notebook
```
